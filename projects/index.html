Stuff I've written that might be of use to someone

  • The Random Coffee Trials Randomiser

    Very basic web app that will accept a list of group members and randomise them into pairs. Will also allow the user to download lists of members/pairs as CSV files so they can be reused and uploaded for future sessions. The code for this is open source (GPL license) and is available on github.
